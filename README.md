# Casa Vacanze Offline (GitHub Pages)

## Pubblicazione (da PC)
1. Apri il repo su GitHub → **Add file → Upload files**
2. **Carica TUTTI i file e la cartella `icons/`** (non caricare lo ZIP come file dentro al repo).
3. Controlla che nella lista file ci siano:
   - `index.html`
   - `service-worker.js`
   - `manifest.webmanifest`
   - `logo.jpg`
   - cartella `icons/`
   - `.nojekyll`
4. Vai su **Settings → Pages**
   - Source: `Deploy from a branch`
   - Branch: `main` / Folder: `/(root)`
   - Salva

## Link da aprire su iPhone/iPad (Safari)
Usa sempre **HTTPS** e questo formato:
`https://<TUO-UTENTE>.github.io/<NOME-REPO>/`

Esempio:
`https://valentinigiorgio1966-eng.github.io/casa-vacanze-offline/`

## Installazione come app (offline)
Safari → Condividi → **Aggiungi alla schermata Home**.
Poi prova Modalità aereo: deve aprirsi comunque.
